# Django Backend Overview

## Purpose
The Django backend orchestrates database access, business logic,
and reporting services.

## Structure
- core: domain models
- services: business logic
- tests: validation & correctness

## Why Django
- Mature ORM
- Strong ecosystem
- Production-ready patterns

## Database Usage
- PostgreSQL: ORM + raw SQL
- MongoDB: analytical reads

## Outcome
A clean, layered backend aligned with best practices.
