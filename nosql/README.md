# MongoDB – Phase 3 Documentation

This folder contains documentation for the MongoDB (NoSQL) portion of the Heritage Site Information System.

## Contents
- NoSQL overview
- ETL design decision
- Query patterns
- Performance and indexing
- SQL vs NoSQL comparison

## Execution Flow
1. MongoDB collections are created
2. Data is seeded from PostgreSQL via ETL
3. MongoDB scripts perform CRUD, analytics, and indexing

MongoDB does not replace PostgreSQL. It complements it.
