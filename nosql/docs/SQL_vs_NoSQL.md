# SQL vs NoSQL Comparison

## PostgreSQL
Used for:
- Strong consistency
- Transactions
- Normalised relational data
- Constraints and integrity

---

## MongoDB
Used for:
- Flexible schemas
- Embedded documents
- Aggregations
- Analytics
- Read-heavy workloads

---

## Architectural Decision
Both databases are used together (polyglot persistence), each playing to its strengths.

This approach improves scalability and maintainability.
